# step1 语义规范
**1.1.** MiniDecaf 的 int 类型具体指 32 位有符号整数类型，范围 [-2^31, 2^31-1]，补码表示。

**1.2.** 编译器应当只接受 [0, 2^31-1] 范围内的整数, 我们未来会支持负数。
     如果整数超过此范围，编译器应当报错。
