# step10 语义规范
**10.1.** 全局变量可以被初始化，但初始值（initializer）只能是整数字面量。
> 严格的说，初始值的值是常量即可，例如可以 `int a=2+3;`，编译器会计算 `2+3==5`。
> 为了简单我们不要求，只要求 `int a=5;` 即可。

**10.2.** 如果没有显式给出初始值，全局变量初始值为 0。

**10.3.** 不允许重复声明全局变量。
> 其实 C 是允许全局变量重复声明，只要不重复定义的。
> `int a; int a=2; int a; int main(){return a;}` 是合法的 C。
> 为了简单，我们不支持这点。
