# 实验指导 step1：一个仅有 return 的 main 函数

实验框架已经完成并通过了本节的测例，因此你不需要在这个阶段修改代码，但需要在 stage 1 的报告中（注意不是 stage 0）完成[通过例子学习](./example.md)一节末尾的思考题。
